PetSpeak

This project focuses on analyzing pet emotions through audio recordings by leveraging techniques from signal processing and machine learning. The system captures sounds made by pets such as barking, meowing, or whining—and extracts meaningful audio features like pitch, frequency, intensity, and spectral patterns. These features are then processed using classification models to identify emotional states such as happiness, stress, aggression, or discomfort.
